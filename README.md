# Instagram Feed Clone

Aplicação web que simula a interface de um feed do Instagram, desenvolvida com React com o objetivo de praticar conceitos fundamentais da biblioteca e a criação de interfaces interativas.

## Sobre o projeto

O projeto reproduz elementos característicos da rede social, incluindo uma barra de stories (apenas visual) e publicações com interações de curtida e salvamento. O foco principal foi o aprendizado de componentes, gerenciamento de estado e manipulação de eventos no React.

## Funcionalidades

- Exibição de um feed de publicações;
- Barra de stories com finalidade visual;
- Curtir e descurtir publicações;
- Salvar e remover publicações dos salvos;
- Atualização dinâmica da interface sem recarregar a página.

## Tecnologias utilizadas

- React
- JavaScript
- HTML5
- CSS3

## Conceitos praticados

- Criação e reutilização de componentes;
- Uso de props para compartilhamento de dados;
- Gerenciamento de estado com Hooks (`useState`);
- Renderização dinâmica de elementos;
- Manipulação de eventos em aplicações React.

## Objetivo

Este projeto foi desenvolvido como parte dos estudos em React, com foco na construção de interfaces modernas e interativas, aplicando conceitos essenciais para o desenvolvimento de aplicações front-end.
